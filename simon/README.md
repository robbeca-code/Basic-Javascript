# Simon Game

### 게임방법
1. POWER을 클릭해서 시작할 준비를 합니다. <br/>
1-1. STRICT는 게임에서 틀렸을 때 다시 처음으로 초기화되는 옵션입니다.
2. Start 버튼을 클릭해서 게임을 시작합니다.
3. 총 6번의 게임이 진행되고 다 맞췄다면 게임에서 승리하게 됩니다.
