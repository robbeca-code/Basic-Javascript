# Memory Game

### 게임방법
1. 카드를 뒤집으면서 맞는 짝을 찾습니다.
2. **Score**가 6이 되면 User의 승리입니다.
3. Restart 버튼을 누르면 게임을 재시작 할 수 있습니다.
